# Stock Price Trend Forecaster

## Project Overview
An end-to-end web application that predicts future stock price trends using Machine Learning. The app fetches real-time data from Yahoo Finance, processes it using Pandas, and utilizes a Random Forest Regressor to forecast stock movements.

## Key Features
- Real-time Data: Live integration with YFinance API.
- Machine Learning: Random Forest algorithm for price prediction.
- Interactive UI: Built with Streamlit for easy user interaction.
- Technical Analysis: Includes Moving Averages (MA5, MA10) as key indicators.
